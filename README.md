# AI Data Analyst Assistant

This project is part of my learning journey into building intelligent AI applications with Pydantic AI. its an experiment of a system where an AI agent can help analyze uploaded datasets through conversation.

## Why?

I created this project to explore:

- how to use Pydantic AI for building an agentic application
- how to connect AI reasoning with data processing tools
- how to combine FastAPI, React, and Python into a complete experience

## What This Does

The app lets a user:

- upload a dataset
- inspect dataset profiles and summaries
- chat with an AI assistant about the data
- ask for analysis, explanations, and insights

It is designed as a learning-focused example of how AI can support data exploration in a conversational interface.

## Tech Stack

- Backend: FastAPI, Pydantic AI, Pandas, DuckDB
- Frontend: React, Vite, CSS
- Data handling: CSV upload and dataset profiling
- Visualization support: Plotly-style data analysis workflows

## Learning Goals

This project helped me practice:

- designing AI agents with tools
- working with structured inputs and outputs using Pydantic
- building API endpoints for AI interactions
- integrating frontend and backend into one experience
- creating a portfolio project that reflects both technical growth and problem-solving mindset

## Project Structure

- backend: FastAPI server, agent logic, dataset handling, and analysis flow
- frontend: Simple React app for interacting with the assistant
- uploads: sample datasets used during development


## Future Improvements

Possible next steps include:

- richer data visualizations
- Adding SQL-based analysis support
- better prompting and agent workflows
- deployment to a cloud environment
